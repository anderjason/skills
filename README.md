# Skills

A small, curated collection of agent skills created and maintained by Jason Anderson.

This repository is the canonical public source for released skills. Each skill is a standalone directory under `skills/` with a `SKILL.md` file and any resources it needs.

The collection follows the portable Agent Skills format. Skills are tested with Codex and Claude Code unless a skill says otherwise. Other compatible agents are community-supported.

## Released skills

- **adversarial-review-loop**: Stress-test a plan, design, or repo with cold-start reviewer subagents that interrogate it one question at a time until it breaks or earns trust.
- **hackathon**: Turn a rough direction into a winning hackathon project, shaped around a peak demo moment, built by a subagent team, and delivered as a real, rehearsed demo.
- **humane-minimalism**: Design or refine interfaces in the humane-minimalist tradition, calm typographic surfaces with deep, tactile behavior.
- **saying-things-simply**: Edit or write prose so it feels simple to the reader, with concrete procedures for cutting, restructuring, and knowing when to stop.
- **writing-for-audio**: Write and edit prose meant to be heard rather than read, from podcast scripts and narration to spoken essays and talks.
