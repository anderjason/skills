# Public skills repository

This repository contains only skills approved for public release. The private authoring workspace is authoritative; releases flow one way into this repository.

Do not copy private workspace history, configuration, automation, operator details, machine paths, or undeclared files into this repository.

Every skill must remain usable as a standalone directory. Keep `SKILL.md`, referenced resources, examples, compatibility notes, attribution, and license information consistent.

Before accepting a release, verify that every file has passed content, privacy, provenance, and license review. Public contributions must not synchronize automatically into any private repository.
